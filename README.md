📌 Descripción del proyecto

Este proyecto implementa el algoritmo de cifrado César mediante una aplicación de escritorio desarrollada en Java utilizando Swing para la interfaz gráfica.

El programa permite al usuario ingresar un mensaje de texto y aplicar un desplazamiento sobre las letras del alfabeto para cifrar o descifrar el contenido. También incluye funcionalidades interactivas para facilitar la selección del desplazamiento y visualizar los resultados en tiempo real.

⚙️ ¿Para qué sirve?

Este programa sirve para:

Cifrar mensajes usando el algoritmo César
Descifrar textos previamente codificados
Aprender los fundamentos básicos de la criptografía clásica
Practicar el manejo de cadenas y lógica en Java
🧠 ¿Cómo funciona?

El algoritmo César consiste en desplazar cada letra del mensaje original un número fijo de posiciones dentro del alfabeto.

Por ejemplo:

Texto original: HOLA
Desplazamiento: 3
Resultado: KROD

🖥️ Características
Interfaz gráfica amigable con Java Swing
Entrada de texto para el mensaje
Botones para cifrar y descifrar
Uso de desplazamiento configurable
Soporte para letras mayúsculas y minúsculas
🛠️ Tecnologías utilizadas
Java
Swing (GUI)
NetBeans (estructura del proyecto)

Ejecutar la clase principal (msj.java)
